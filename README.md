# hexo-theme-spa

Hexo 单页应用主题，Vue3 + Tailwind CSS。

练手，仅在功能上初步实现了基于 Vue3 的单页应用，SSR、SSG 技术的实践。

# Note

- Should restart `hexo server` if update `_config.yml`
- `Node >= 12`
- 一些 `after_post_render` 处理 HTML 的 filter 并不会有效果
